----------------------------------------------

Můj projek o Medlovu a jeho zajímvasoteh, jako jsou okolní obce, cesta do Medlova, něco z málo o Medlovu a následně kontaktní údaje na nás.

----------------------------------------------

Použité funce:

HTML - jazyk v kterém je vytvořený základ mé stránky
PHP - využití pro komunikaci mezi soubory (header, footer, odkazy) /templates/... + funkce v /.php/... (soubory s příponou .php) 
CSS - využití ve stylování, designu a celkovém vzhledu a uspořádání stránky - /.css/styles.css
JS - využití při vytváření vyskakujích oken s úvodním textem při přejití na stránku, následně při kliknutní na stránku "Fotbal" se vám oběví okno s cookies, které si zapamatuje vaše jméno, a pak tlačítko "jaký je den" které vypíše jaká je dnes den a čas, soubory se nachází v /js/... a jeden script přímo v souboru.

----------------------------------------------

Využitý software:

Program - Visual Studio Code
GitHub desktop - pro download a upload změn v projekt na GitHub
GitHub - zde se nachází kód mé stránky
InfinityFree - hosting stránky

Struktura stránky:

STRÁNKA NENÍ RESPONZIVNÍ NA MOBILU A NA NĚKTERÝCH ROZLIŠENÍCH !!!

Zbytek stránky - zde se nachází orámovaný text společně s obrázky
